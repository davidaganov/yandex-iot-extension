# Умный дом

Открывает [yandex.ru/iot](https://yandex.ru/iot/) во всплывающем окне расширения.

Только **Chrome / Edge / Chromium** (ручная установка).

## Установка

1. Скачайте или склонируйте репозиторий.
2. Откройте `chrome://extensions` (или `edge://extensions`).
3. Включите **Режим разработчика**.
4. **Загрузить распакованное расширение** → выберите папку репозитория.
5. Закрепите иконку на панели и нажмите на неё.

После правок кода нажмите **Обновить** на карточке расширения.

## Что делает расширение

- Popup с iframe официального сайта
- Снимает запрет на встраивание (CSP / X-Frame-Options) через DNR
- Content script: компактный UI

## Файлы

| Файл                       | Назначение          |
| -------------------------- | ------------------- |
| `manifest.json`            | Манифест MV3        |
| `popup.html` / `popup.css` | Окно расширения     |
| `rules.json`               | Правила DNR         |
| `content/`                 | Правки страницы IoT |
| `icons/`                   | Иконки              |
